# Topology V2 — Keyboard Shortcuts Reference

## Navigation
| Shortcut | Action | Context |
|----------|--------|---------|
| 1 | Cluster View | Switch to cluster overview |
| 2 | Namespace View | Switch to namespace-scoped view |
| 3 | Workload View | Switch to workload grouping |
| 4 | Resource-Centric View | Switch to BFS exploration |
| 5 | RBAC View | Switch to RBAC relationships |
| Backspace | Navigate Back | Return to previous view in history |
| Escape | Back / Deselect / Close | Context-dependent: closes panel, deselects node, or navigates back |

## Viewport
| Shortcut | Action |
|----------|--------|
| F | Fit to Screen |
| + / = | Zoom In |
| - | Zoom Out |
| M | Toggle Minimap |

## Toggles
| Shortcut | Action |
|----------|--------|
| E | Toggle Edge Labels |
| H | Toggle Health Overlay |
| C | Toggle Cost Overlay |

## Actions
| Shortcut | Action |
|----------|--------|
| / | Focus Search Input |
| S | Screenshot to Clipboard (PNG) |
| ? | Show This Shortcuts Reference |

## Mouse Interactions
| Action | Effect |
|--------|--------|
| Click node | Select and show detail panel |
| Double-click node | Navigate into / re-center BFS |
| Click canvas | Deselect current node |
| Drag canvas | Pan the viewport |
| Scroll wheel | Zoom in/out |
| Hover edge label | Show relationship detail |

## Notes
- Shortcuts are disabled when typing in the search input
- Modifier keys (Ctrl/Cmd/Alt) are not required for any shortcut
- All shortcuts follow the PRD Section 8.2 specification
